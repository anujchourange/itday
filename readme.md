# ITDAY assignment

This is rest api backend app using NodeJS, ExpressJS and MongoDB.

## Usage

Rename the .env copy to .env and add your MONGO_URI and JWT_SECRET

### Install dependencies

```
# Backend deps
npm install
```

### Run Server

```
npm run dev
```
